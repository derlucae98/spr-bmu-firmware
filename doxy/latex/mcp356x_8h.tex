\doxysubsection{Detailed Description}
Library to interface with MCP356x(\+R) ADCs. It is fully hardware independent and uses callback functions to interface with the hardware. This library can be used with multiple ADCs which are addressed by objects of type \mbox{\hyperlink{structmcp356x__obj__t}{mcp356x\+\_\+obj\+\_\+t}}.
